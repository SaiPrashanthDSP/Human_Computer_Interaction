How to Use

  • Click the Start Recognition button.
  • Speak one of the commands listed below.
  • Receive immediate feedback and see the action performed.

Available Commands

  • Scroll up - Scrolls the page upward.
  • Scroll down - Scrolls the page downward.
  • Go back - Navigates to the previous page.
  • Reload page - Refreshes the page.
  • Search for [term] - Opens a Google search for the term.
  • Open [website] - Opens a specified website (e.g., "Open YouTube").
  • Open New Tab - Opens a new browser tab.
  • Close Tab - Closes the current tab.
  • Zoom In - Zooms in on the page.
  • Zoom Out - Zooms out on the page.
  • Go to Homepage - Takes you to the homepage.
  • Scroll to Top - Scrolls the page back to the top.
  • Maximize Window - Maximizes the browser window.
  • Show Time - Displays the current time.
